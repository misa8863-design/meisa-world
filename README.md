## noteリンクの変更

`index.html` の `<body>` にある `data-note-url` を変更してください。

```html
<body data-note-url="https://note.com/meisa_haiagari">
```

## 公開

このフォルダの中身をGitHubへアップロードすると、Vercelなどで静的サイトとして公開できます。
